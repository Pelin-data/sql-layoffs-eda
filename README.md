# Global Layoffs Exploratory Data Analysis (SQL)

## Project Overview
This project explores global company layoffs between 2020 and 2023 using SQL.
The goal is to identify trends across time, industries, countries, and company stages.

## Tools Used
- MySQL
- SQL (CTEs, Window Functions, Aggregations)

## Key Analyses
- Total layoffs by company
- Layoffs by year and month
- Rolling (cumulative) layoffs over time
- Top 5 companies with highest layoffs per year
- Layoffs by industry, country, and company stage
